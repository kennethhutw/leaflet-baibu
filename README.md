# leaflet-baibu
Leaflet plugin that enables the use of baibu Map tiles

## Usage

`   var map = L.map('container');
	map.setView([39.916527,116.397128], 11);
	var qqLayer = new L.BMap();
    map.addLayer(qqLayer);` .
